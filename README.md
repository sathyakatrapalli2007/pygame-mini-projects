# 🎮 Pygame Mini Projects

This repository contains small Pygame-based projects built while learning core concepts of game development and animation. Each project focuses on a specific idea such as movement, layering, or visual effects.

---

## 🌧️ Rain Simulation

A continuous rain effect built using a grid of raindrop sprites.

Each vertical column is assigned a different speed, and drops are slightly offset in position to avoid uniform patterns. When a drop leaves the screen, it is repositioned above, creating a seamless looping animation.

This project focuses on how variation (speed, spacing, randomness) can make simple motion look more natural.

---

## ✨ Star Shining Animation

A simple star field where stars appear to “shine” by alternating between two layers.

Two complete grids of stars are created — a normal layer and a brighter one. A timer switches between them at fixed intervals, producing a flickering effect without modifying individual stars.

This project explores how layering and timing can be used to simulate visual effects.

---

## 🧠 What I focused on

* Working with sprite groups in Pygame
* Managing multiple objects using grid-based layouts
* Creating continuous animations
* Using randomness to improve realism
* Building visual effects with simple logic

---

## ▶️ Running the projects

Each project can be run independently:

```bash
cd rain_simulation
python main.py
```

```bash
cd star_animation
python main.py
```

---

## 📌 Notes

These are small, focused experiments rather than full games. The goal was to understand how motion, layering, and variation affect what we see on screen, and to build those ideas from scratch.
